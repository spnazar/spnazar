# IT Engineer & Backend Developer

Специализируюсь на проектировании отказоустойчивых систем и разработке бизнес-ориентированных сервисов. Сочетаю опыт системного администрирования корпоративного уровня с разработкой на PHP (Laravel) и Node.js.

---

### 🌐 Infrastructure & Networking
*   **Administration:** Экспертное управление **Windows Server** и **Linux** (Debian/Ubuntu/CentOS).
*   **Directory Services:** Глубокая настройка **Active Directory** (GPO, управление правами, политики доступа).
*   **Networking:** Проектирование сетей (LAN, VPN), работа с оборудованием **UniFi**, сегментация и контроль трафика.
*   **Security:** Настройка и поддержка корпоративных **VPN**-решений для удаленной работы.

### 💻 Software Development
*   **Backend:** Разработка CRM/ERP систем на **Laravel** и **Node.js**, проектирование сложных REST API.
*   **Database:** Оптимизация SQL-запросов, администрирование **PostgreSQL**, работа с архитектурой данных.
*   **Automation:** Написание скриптов автоматизации на **Python**, работа с очередями задач и асинхронными процессами.
*   **Hardware:** Опыт разработки IoT-устройств на базе **ESP32** и прототипирование в Tinkercad.

### 🛠 Management & Operations
*   **Service Desk:** Работа в **JIRA**, выстраивание процессов обработки инцидентов и контроль **SLA**.
*   **Support:** Глубокая диагностика инцидентов 1-2 линии, устранение критических сбоев в инфраструктуре.
*   **Tools:** Antigravity, RustDesk, Webpack, Git.
